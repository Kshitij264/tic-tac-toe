# Tic Tac Toe Game 🎮

A responsive Tic Tac Toe game with two game modes:

- **Player vs Player**
- **Player vs AI (using Minimax algorithm)**

Includes a theme toggle for **dark/light mode** and a smooth, intuitive UI for easy interaction.
